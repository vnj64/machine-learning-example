## Пункты к выполнению

```
Выбор набора данных: Выберите набор данных для классификации или регрессии. Для этого задания возьмем набор данных о прогнозировании оттока клиентов, который вы предложили в пункте 1c (i).
```
```
Импорт данных и предварительный анализ:
a. Загрузите данные и оцените типы данных, наличие дубликатов и пропущенных значений.
b. Выполните исследовательский анализ данных (EDA) и создайте информативные графики.
c. Оцените связь признаков с целевой переменной.
```
```Предобработка данных:
a. Примените различные методы предобработки, такие как стандартизация, нормализация, кодирование категориальных признаков и другие, в зависимости от требований данных.
b. Учтите начальное распределение данных при применении методов.
```
```
Разделение данных на тренировочную и валидационную выборку: Определитесь с метриками, по которым будете оценивать модель.
```
```
Выбор моделей:
a. Выберите две модели машинного обучения, например, линейную модель (как baseline) и какую-то более сложную модель.
```
```
Обучение линейной модели (baseline):
a. Обучите линейную модель без дополнительных преобразований.
```
```
Улучшение качества модели:
a. Для числовых признаков можно попробовать полиномиальные признаки.
b. Рассмотрите добавление категориальных признаков и другие возможные улучшения.
c. Рассмотрите изменение целевой переменной (например, веса классов для баланса).
```
```
Оптимизация гиперпараметров моделей:
a. Используйте методы оптимизации гиперпараметров, такие как GridSearch, RandomizedSearch или Optuna.
b. Определите, какие гиперпараметры дали наилучшие результаты.
```

```
Оценка моделей на тестовой выборке:
a. Оцените модели на тестовой выборке.
b. Проанализируйте ошибки моделей и попытайтесь понять, почему они принимают те или иные решения.
```
```
Сравнение моделей:
a. Сравните модели между собой по метрикам, распределению ошибок и т.д.
```
